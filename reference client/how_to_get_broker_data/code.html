How to Get Broker Data

Follow this step-by-step guide to extract your transaction history JSON directly from your broker's dashboard for accurate analysis.

01

Log in to Broker Dashboard

Navigate to your broker's website and log in to your account. Go to the main dashboard or "Account Overview" page where your portfolio summary is visible.

02

Open Network Tab

Right-click anywhere on the page and select Inspect, or press F12. In the Developer Tools window, click on the Network tab.

03

Filter and Refresh

In the Network tab filter box, type history or json. Refresh the page (F5). You should see a request appear in the list.

04

Verify and Copy Response

Click on the request name, then select the Response tab. Ensure the data structure looks like the example below, then copy the entire JSON object.

code Expected JSON Format 
{
  "account_id": "882910-X",
  "history": [
    {
      "symbol": "AAPL",
      "order_type": "BUY",
      "quantity": 150,
      "price": 142.50,
      "timestamp": "2023-10-24T14:30:00Z",
      "status": "FILLED"
    },
    {
      "symbol": "TSLA",
      "order_type": "SELL",
      "quantity": 50,
      "price": 210.25,
      "timestamp": "2023-10-25T09:45:00Z",
      "status": "FILLED"
    }
  ]
}

Once you have verified the data matches the format above, proceed to import.
